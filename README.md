# Spring-Boot-MVC-JPA
It's a starting project for spring *Boot with spring *MVC and spring data *JPA (in memory data base *H2)

**Technologies/Spring Modules:**
* Spring-Boot
* Spring-MVC
* Spring-Data-JPA
* Lombok (	*@Data,	*@Log)
* ApplicationRunner
* H2 (In Memory DataBase, Console also available)
* BootStrap (CSS,JS)
* JSP
* JSTL
